This project explores the impact of feature selection on model performance using a diabetes dataset from Kaggle.

A Random Forest Classifier is applied to predict diabetes outcomes based on medical attributes such as glucose level, BMI, and age. The experiment focuses on varying the number of selected features (top_k = 3, 5, 6) using feature importance scores.

For each configuration, the model is trained and evaluated to compare:
 • Accuracy performance
 • Most influential features

The results highlight how reducing or increasing the number of input features affects prediction accuracy and model behavior.
